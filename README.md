# ExplainXKCD-Button
Greasemonkey userscript that changes [XKCD](https://www.xkcd.com/) store button to an 'explain' button which links to the relevant [ExplainXKCD](https://www.explainxkcd.com/wiki/index.php/Main_Page) page.
